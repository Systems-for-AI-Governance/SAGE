Getting Started

SAIGRC is an open-source platform designed to help healthcare organizations manage AI governance, risk, and compliance. This guide will help you get started with implementing SAIGRC in your organization.

Prerequisites

  • Python 3.8 or higher
  • Docker (recommended)
  • PostgreSQL database

Installation Guide

You can install SAIGRC using Docker or from source:

# Using Docker
docker pull saigrc/platform
docker-compose up -d
# From source
git clone https://github.com/Systems-for-AI-Governance/saig-health.git
cd saig-health
pip install -r requirements.txt

Configuration

Configure SAIGRC by editing the config.yaml file:

database:
  host: localhost
  port: 5432
  name: saigrc

security:
  enable_audit_logs: true
  encryption_key: your-secret-key

API Reference

SAIGRC provides a RESTful API for integration with your existing systems:
